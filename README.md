 Flipkart Clone Web Application
📌 Project Overview

This project is a Flipkart-like product listing webpage clone built using HTML, CSS, and JavaScript.
It replicates the UI of an e-commerce platform where users can browse products, view details, and explore filtering options.

🎯 Objectives

Create a responsive e-commerce UI

Understand product listing layout

Implement sidebar filters

Practice frontend development using HTML, CSS, JS

Improve UI/UX design skills

🛠️ Technologies Used

HTML – Structure of the webpage

CSS – Styling and layout design

JavaScript – Interactivity

Web APIs – Placeholder images


🚀 Features

✔ Flipkart-style header with search bar
✔ Sidebar filters (Brand, Processor, etc.)
✔ Product listing cards
✔ Price, discount, and rating display
✔ Clean and responsive layout
